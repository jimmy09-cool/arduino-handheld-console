# Arduino Handheld Retro Console 🎮

A portable Arduino Mega handheld game console using a MAX7219 8x8 LED matrix.

## Features
- Snake
- Pong
- Dodge game
- Menu system
- Sound effects
- EEPROM high score saving

## Hardware
- Arduino Mega 2560
- MAX7219 8x8 LED Matrix
- Joystick module (A0/A1)
- Push button (pin 22)
- Buzzer (pin 8)

## Controls
- Joystick → move
- Button → select / exit

## Games
- Snake: classic growth game
- Pong: single paddle pong
- Dodge: avoid falling obstacles
